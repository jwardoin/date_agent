# dAIte-agent

dAIte-agent is a self-hosted, locally-powered agentic AI used to generate date ideas.

# Requirements

- Python 3.10+

## How To Use

- Use the `sample.env` file to gather the required environmental variables.

- Create a .csv file and add activities, locations, etc. with as little or as much information as you desire. Define the file path in your `.env` file.

- Define your Ollama variables (model, api key, and url if hosting on a remote server)
